Examen Git y Github

Alumn@: [daniel paucar]

  1. Que es un sistema de control de versiones?

    Se llama control de versiones a la gestión de los diversos cambios que se realizan sobre los elementos de algún producto o una configuración del misma

  2. Cual es la diferencia entre git y github?

    Git y GitHub son dos entidades diferentes que te ayudan a administrar y alojar archivos

  3. Cómo creo una rama en git?

    El comando git branch permite crear una rama nueva

  4. Cómo clono un repositorio?

    el comando git clone copia un repositorio de Git existente.
