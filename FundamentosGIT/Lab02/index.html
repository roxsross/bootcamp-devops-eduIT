Examen Git y Github

Alumn@: [Augusto valls ]

  1. Que es un sistema de control de versiones?

    [Los sistemas de control de versiones son herramientas de software que ayudan a los equipos de software a gestionar los cambios en el código fuente a lo largo del tiempo.]

  2. Cual es la diferencia entre git y github?

    [GitHub es una plataforma en línea de alojamiento (o para compartir) repositorios de Git. Nos ayuda crear una avenida para colaborar fácilmente con cualquier persona, en cualquier lugar, en cualquier momento.]

  3. Cómo creo una rama en git?

    [git branch]

  4. Cómo clono un repositorio?

    [git-clone]
