Examen Git y Github

Alumn@: [Nombre Completo]

  1. Que es un sistema de control de versiones?

    [Los sistemas de control de versiones son software que ayudan a realizar un seguimiento de los cambios realizados en el código a lo largo del tiempo.]

  2. Cual es la diferencia entre git y github?

    [Git sirve para controlar las versiones de los archivos mientras que GitHub es una plataforma para alojar tus repositorios Git.]

  3. Cómo creo una rama en git?

    [El comando git branch permite crear una rama nueva.]

  4. Cómo clono un repositorio?

    [El comando git clone copia un repositorio de Git existente.]
